Это моя первая веб-страница, созданная с использованием HTML. Здесь я могу разместить любую информацию, которую захочу.

HTML (HyperText Markup Language) — это язык разметки гипертекста, который используется для создания структуры веб-страниц. С помощью него можно добавлять заголовки, абзацы, списки, ссылки и многое другое.

Чтобы узнать больше о HTML, рекомендую посетить сайт MDN Web Docs. Там вы найдете множество полезных ресурсов и примеров кода.
